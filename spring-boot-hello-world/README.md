# Spring Boot Hello World Example
Get started with the Spring Boot application, a hello world example.

This project demonstrates a basic Spring Boot application with Java 17 and Spring Boot 3.1.2.

## 1. How to start
```bash
$ git clone https://github.com/ramakrishnametla/spring-boot-with-java-17.git

$ cd spring-boot-with-java-17/spring-boot-hello-world

# Tomcat started at 8080
$ mvn spring-boot:run

# test
curl localhost:8080

```
